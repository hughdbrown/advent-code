# Setup
```
# Create a virtualenv for the project
pyenv virtualenv 3.12.0 advent-of-code-3.12.0
pyenv activate advent-of-code-3.12.0

# Make the virtualenv hte default for the directory
pyenv local advent-of-code-3.12.0

# Install the python requirements
pip install --upgrade pip
pip install -r requirements.txt
```
